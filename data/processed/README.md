# processed/

Clean matrices ready for analysis, generated by `data/notebook/prepare_data.qmd`. All files are tab-separated. Samples and metadata rows are in the same order within each cohort.

---

### beat_aml_expression.tsv.xz
Samples × genes expression matrix for BEAT-AML. Restricted to samples with WES data. First column: `sample_id`. Values: log2 CPM-TMM. Gene symbols as column headers (Hugo).

### beat_aml_metadata.tsv
Sample-level metadata for BEAT-AML, one row per sample. Key columns: `sample_id`, `patientId`, `Gender`, `Disease_Type`. Row order matches `beat_aml_expression.tsv.xz`.

---

### tcga_aml_expression.tsv.xz
Samples × genes expression matrix for TCGA-LAML. Restricted to samples with WES data. First column: `sample_id`. Values: linear CPM-TMM. Gene symbols as column headers (Hugo).

### tcga_aml_metadata.tsv
Sample-level metadata for TCGA-AML, one row per sample. Columns: `sample_id`, `Patient_Id`, `Gender`, `Disease_Type`. Row order matches `tcga_aml_expression.tsv.xz`.

---

### ngs_ptl_metabolome.tsv
Samples × metabolites abundance matrix for NGS-PTL (92 samples, ~300 metabolites). First column: `sample_id`. Values: log-transformed, DNA-normalised.

### ngs_ptl_metadata.tsv
Sample labels for NGS-PTL. Columns: `sample_id`, `sample_type` (`AML` or `control`). Controls are healthy CD34+ progenitors.

---

### gem_gene_subsystems.tsv
Flat table linking Human-GEM metabolic genes to subsystems, parsed from GPR rules. Columns: `hugo`, `ensembl_id`, `n_reactions`, `n_subsystems`, `subsystems` (`;`-separated), `reactions` (`;`-separated). Used to filter expression matrices to metabolic genes before WGCNA.
